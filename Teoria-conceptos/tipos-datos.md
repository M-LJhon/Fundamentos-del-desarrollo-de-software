# Tipos de datos
## ¿Qué son los tipos de datos en programación?
Los tipos de datos son una forma de clasificar los valores que puede almacenar una variable, dependiendo el tipo de dato podemos realizar ciertas operaciones u otras.
## Clasificación de los tipos de datos
### Datos primitivos:
* Numéricos:
    * Int.
    * Float.
    * Double.
* Texto:
    * String.
    * Char.
* Booleanos:
    * False.
    * True.
* Null / void.
### Tipos compuestos:
* Arrays.
* Objetos.
* Tuplas.
* Estructuras.
* Enumeraciones.
### Tipos especiales según el lenguaje.
* Date / time.
* Binarios.
* Decimal.
